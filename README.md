## 🚀 Sobre mim

Comecei como Jovem Aprendiz em desenvolvimento web e, em 6 meses, assumi a responsabilidade técnica pelo setor de TI de uma transportadora de grande porte — hoje lidero o desenvolvimento e a infraestrutura por lá.

Tenho experiência liderando projetos complexos, como integrações com sistemas governamentais (SEFAZ) para emissão de CT-e, CTRC e NF-e, aplicando **Domain-Driven Design (DDD)** para isolar regras de negócio tributárias, além do desenvolvimento de PWAs para logística com geolocalização.

Gosto de resolver problemas críticos em produção, criar componentes reutilizáveis e escaláveis, e estou sempre em busca de dominar novas tecnologias.

🔭 Atualmente expandindo horizontes para **N8N**, **Java** e **Vue.js**.
